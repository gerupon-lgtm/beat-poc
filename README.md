# リズム戦闘 PoC

## 実行方法

`index.html`をブラウザで開いてください。音声はブラウザの制約により、「戦闘開始」を押した後に再生されます。

Webサーバーへ配置する場合は、このフォルダーの内容をそのまま公開ディレクトリへ配置してください。

通常URLは正式採用した合成アニメーション表示です。従来のrAF表示を比較する場合は公開URLへ`?visual=raf`を付けてください。Web Animations API非対応環境では自動的にrAF表示へ戻ります。

## 収録ファイル

- `index.html`: 画面本体
- `css/rhythm-battle-poc.css`: 表示スタイル
- `js/rhythm-battle-poc.js`: 戦闘、音声、判定処理

外部ライブラリ、画像、データベース、バックエンドAPIは使用していません。
